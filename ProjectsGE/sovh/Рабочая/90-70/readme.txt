Порядок добавления функции CalcQ в проект:

скопировать файл Calcq.exe на диск С

Открыть LM

F1 открыть проект
F6 Библиотека
F6 добавить элемент в библиотеку
	New element: c:\calcq
	element type: external block
	enter
	number of input: 3
	enter

esc
F3 import to folder
	CALCQ
	enter 
	Y
	

Интерфейс блока Calcq:
Массив входов (X1):
ON_CALC если = 1, то расчет ведется, иначе расход = 0, Real
Nst частота вращения СТ, Real, об/мин
Ps давление на входе компрессора, Real, кгс/см2
Pd давление на выходе компрессора, Real, кгс/см2
Ts температура газа на входе компрессора, грС
Td температура газа на выходе компрессора, грС

Массив выходов (Х2):
Ky запас по помпажу, Real, %
Q_pr приведенная производительность компрессора, Real, м3/мин
Q_kom производительность по паспортной характеристике, Real, млн.м3/сут
Ni внутренняя мощность компрессора, кВт
KPD КПД компрессора
G массовый расход газа, кг/с

Массив констант (X3):
Nnom номинальная частота вращения СТ, Real, 22300 об/мин
Qmin минимальная приведенная производительность компрессора, Real, 12.3 м3/час
p_g плотность газа, Real, 0.98 кг/м3
B барометрическое давление, Real, 750 мм.рт.ст
